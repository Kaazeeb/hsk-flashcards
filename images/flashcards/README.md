# Image flashcards

Place small image assets here for the v35 Image cards page.

Recommended:
- WebP or SVG.
- 256x256 or 384x384 for raster images.
- Keep each image under about 50 KB when possible.
- Reference paths relative to this folder in `js/image-cards-data.js`.

Example path in data:

```js
imagePath: "kids/animals/cat.webp"
```

Corresponding file:

```text
images/flashcards/kids/animals/cat.webp
```
