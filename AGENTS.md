# Repository Guidance for AI Coding Agents

- The repository standard language is English. Use English for documentation, comments, pull request text, commit messages, and user-facing copy added to the project unless an existing file or explicit task requires another language.
- Treat user questions as questions that must be answered directly. A question is not a suggestion or instruction to take action; do not act on a question unless the user explicitly asks you to perform an action.
- Be transparent about constraints, missing context, tool failures, or uncertainty. If you are blocked from completing a task because required information cannot be accessed, extracted, verified, or inferred reliably, report the blocker clearly instead of inventing, approximating, or fabricating content.
- Never create fake or partially invented deliverables to appear to satisfy a request. If a task cannot be completed with the available context and tools, stop and ask the user for the missing information, an alternate source, or permission to take a different approach.
